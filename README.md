# textbackend

testbackend used to create a row/level organized content tree.
the tree looks like this:

    content[
      { ... }
      { level: 0, text: "some words at one line of the document" }
      { level: 1, text: "the level can be used to tab one inside" }
      { level: 1, text: "..." }
      { ... }
    ]
